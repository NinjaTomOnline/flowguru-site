# FlowGuru Public Site

Static marketing, support, privacy, FAQ, press, and release-note pages for FlowGuru.

## Published URLs

- Marketing: `https://ninjatomonline.github.io/flowguru-site/`
- Support: `https://ninjatomonline.github.io/flowguru-site/support.html`
- Privacy: `https://ninjatomonline.github.io/flowguru-site/privacy.html`

## Publish

This repo can be served from GitHub Pages using the `main` branch root. It also includes a Pages workflow for artifact-based deploys if that mode is preferred later.

The source of truth lives in the private FlowGuru app repo at `Release/HostedPages/`.
