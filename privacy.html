Privacy

Clear data boundaries for early access.

Last updated: April 30, 2026. FlowGuru is building a flow arts community platform. This policy covers the current account, profile, support, and staging operations foundation.

Data FlowGuru collects

  • Account details such as email address, display name, and password credentials stored as secure password hashes.
  • Profile details you choose to save, including bio, location, disciplines, skill level, goals, avatar URL, and profile photo uploads when media storage is configured.
  • Session and security data such as secure cookies, request IDs, IP address, user agent, and rate-limit signals.
  • Transactional email events needed for email verification and password reset delivery.
  • First-party onboarding analytics events such as profile setup started, sign-in mode selected, profile saved, and feedback submitted.

How data is used

FlowGuru uses data to create and protect accounts, show and update profiles, send verification and password reset emails, understand where early testers get stuck, and operate staging safely through logs, health checks, backups, and abuse prevention.

Onboarding analytics

FlowGuru uses a small first-party analytics endpoint for early-access onboarding only. Events are limited to an allowlist, stored in FlowGuru-owned Postgres, and used to improve account, profile, verification, and feedback flows.

FlowGuru does not load third-party analytics scripts, ad pixels, heatmaps, session replay tools, or cross-site tracking. Analytics events must not include passwords, reset tokens, verification tokens, profile free text, feedback message bodies, API keys, raw IP addresses, full user agents, or full URLs with query strings. The early staging retention target is 90 days.

Service providers

Current early-access providers include Hostinger for the VPS, Backblaze B2 for encrypted database backup storage and staging avatar storage, and Resend for transactional account email.

What is not active yet

The current product does not include payments, marketplace listings, public feeds, tutorial uploads, event management, or moderation workflows. Avatar uploads are limited to profile photos. FlowGuru also does not use third-party advertising or cross-site tracking.

Your choices

You can update your profile from the FlowGuru app. Signed-in users can request account deletion from account settings after confirming their account email and current password. You can also email support@flowguru.app from the account email with the subject "Delete my FlowGuru account."
